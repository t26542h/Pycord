# PyCord Lite

Легкий Discord-подобный прототип на Python: вход по нику без регистрации, серверы, текстовые и голосовые каналы, микрофон и демонстрация экрана через WebRTC.

## Запуск

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
python app.py
```

Откройте:

```text
http://127.0.0.1:8000
```

## Как проверить с друзьями в одной сети

Самый простой способ на Windows:

```powershell
.\start-lan.bat
```

Если PowerShell/Windows спросит про брандмауэр, разрешите Python доступ в частной сети.

После запуска программа покажет адреса вида:

```text
http://192.168.1.23:8000
```

Именно этот адрес нужно отправлять людям в вашей Wi-Fi/LAN сети. Не отправляйте `http://127.0.0.1:8000`: для другого человека это его собственный компьютер, а не ваш сервер.

Ручной запуск:

```powershell
$env:PYCORD_HTTP_HOST="0.0.0.0"
$env:PYCORD_WS_HOST="0.0.0.0"
python app.py
```

На другом устройстве откройте `http://IP_ХОСТА:8000`.

Браузеры разрешают микрофон и демонстрацию экрана только в защищенном контексте. На том же компьютере `http://127.0.0.1:8000` подходит, а для других устройств чаще понадобится HTTPS.

Для интернета понадобится публичный хостинг, домен с HTTPS и TURN-сервер для стабильного WebRTC за сложными NAT.
